Project information

  • Project: Loan Fraud Prediction using Machine Learning - Develop a robust machine learning (ML) model to predict loan applicant creditworthiness, mitigating the risk of loan defaults and fraudulent activities.
  • Business goal : The primary objective of this project was to develop a robust and efficient machine learning model using LightGBM to achieve an accuracy of at least 80% in predicting loan repayment probabilities, thereby mitigating the risk of loan defaults for financial institutions. This involved preprocessing and analyzing a dataset containing over 233,000 records and 40 features, performing detailed feature selection, and training the model to ensure high prediction reliability. The goal was to streamline the loan approval process, reducing the manual workload by 50% for banking officials.
  • Technology: Python, Machine Learning (Regression and Prediction), PowerBI for Data visualization, SQL, Microsoft office suite.

Role and Contributions

Role: Scrum master, Developer, Project documentation controller.

  • Developed a machine learning model using LightGBM for predicting loan repayment probabilities, which increased prediction accuracy to 80%, significantly reducing the risk of loan defaults for potentially partnering banks in the future.
  • Implemented data preprocessing techniques, including feature selection and scaling, which streamlined the dataset and improved model training efficiency, cutting down processing time by 30%.
  • Designed comprehensive project documentation and user manuals that facilitated easy adoption and understanding of the machine learning system, thereby reducing training time for bank officials by 40%.
  • Consulted with bank officials and financial stakeholders to tailor the machine learning model to specific banking needs, enhancing decision-making accuracy and contributing improvement in customer satisfaction and loan processing speed.
